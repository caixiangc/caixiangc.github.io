写博客的时候注意事项和步骤

发表于 2017-11-03 | 分类于 RecordLife |
用Typora编译器本地编写md文件

在编写的时候，要在文章头加入：

  • categories: c # 配置categories
  • title: title_test
  • — //这个分隔线很重要！！！如果漏了无法分页

如果要给博客添加分类，那么要在theme文件夹下的next中的配置文件中添加，这里的添加的分类必须和前面的categories: c # 配置categories保持一直。

在文章写好，生成.md 文件后，放入hexo 的sourse文件夹下的_posts文件夹里面，启动git-bash

TIP：最后注意如果文章目录有 record_life 有下划线这种命名，那么在hexo g 生成的时候会自动编译出record-life的文件夹所以会出现:hexo Cannot GET /categories/Record_life/ 这样的报错，解决方案：在public/categories中修改之前的文件夹
